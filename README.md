### Stack
* React - Mobx
* Google API Translation & Cloud Vision
* Webpack
* Ngrok

### Development

To start a development on default port 3100:
* ```$ npm install```
* ```$ npm start```
* ```$ ngrok tcp 3100```

Build:
* ```$ npm run build```
